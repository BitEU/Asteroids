# Asteroids
**A console version of Asteroids written in C#**

I made this in three days so it's a bit messy, but I'm quite happy with it.


**CONTROLS**

* W or UP    : Accelerate
* A or LEFT  : Rotate to the left
* D or RIGHT : Rotate to the right


* Escape     : Exit
* P          : Pause
